Je bevind je nu bij kamer 3a140, een van de spreekkamers die gebruikt worden door mensen van het zorgteam en passend onderwijs.


Vraag:
Je hebt als student bij de ICT Academie verschillende websites nodig.
  1. een website waar je kunt aangeven dat je ziek bent
  2. een website waar al je opdrachten staan en waar je docenten bijhouden wat je hebt ingeleverd
  3. een website voor het bijhouden van gewerkte uren op stage
  4. een website waar je je talen oefent
  5. een website waar je gratis en legaal alle microsoft operating systems kunt downloaden
  6. (alleen voor applicatieontwikkelaars) een website voor het maken van lijstjes
  7. (alleen voor applicatieontwikkelaars) een website voor het bewaren van alle versies van zelfgeschreven programmacode
  8. (alleen voor applicatieontwikkelaars) een website waar van elke programmeer opdracht de functionele en technische eisen staan en soms voorbeeldcode

Noteer voor elke site de URL.

De volgende code vind je bij bij de studentenadministratie (begane grond, tegenover de kantine).
